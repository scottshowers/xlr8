## ğŸ”„ How Foundation Intelligence Differs from Project Files

### Foundation Intelligence (Sidebar)
- **Scope:** ALL projects
- **Purpose:** Train local LLM
- **Content:** General knowledge, best practices, standards
- **Examples:** UKG guides, industry regulations, templates
- **Usage:** LLM references automatically
- **Location:** Shared across entire application

### Project Files (Main Content)
- **Scope:** Single project/customer
- **Purpose:** Parse and analyze customer data
- **Content:** Customer-specific files
- **Examples:** Pay registers, employee rosters, org charts
- **Usage:** Direct processing and analysis
- **Location:** Isolated per project

**Think of it as:**
- Foundation Intelligence = "What the LLM knows"
- Project Files = "What you're working on"
